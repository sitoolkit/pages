SI-Toolkit for Application Development

SI-Toolkit for Application Development(SIT-AD)はJava EE WebアプリケーションのMavenアーキタイプです。 DBMSやAPサーバー、DBマイグレーションツール、自動テストツールが、インストール不要ですぐに使える状態でセットアップされています。 ツール群はMavenコマンド1つで実行できるため、JenkinsなどのCIツール上で動作させるのも簡単です。 SIT-ADの詳細な情報はこちらを参照してください。

SIT-ADの詳細

5分でプログラミング開始

SIT-ADのアーキタイプはミドルウェアのセットアップスクリプトやJSF、JPAの設定を同梱しているため、 環境構築はすぐに終わりソースコードの実装に着手できます。
  • DB2
  • Derby
  • MySQL
  • Oracle DB
  • Payara
  • WebSphere Liberty
  • Wildfly

主要なミドルウェアに対応

Java EEのアプリケーションで使用する主要なDBMS、APサーバーに対応しています。

強力な開発ツールがセットアップ済み

ソースコード解析、End-to-End自動テスト、DBマイグレーションなど、 省力化に欠かせない開発ツールがセットアップ済ですぐに使えます。

CI / CD First

ミドルウェアや開発ツールの操作はコマンド1つで実行できるため、 JenkinsなどのCIツールでビルドパイプラインがすぐに構築できます。

SI-Toolkit for Web Testing

SI-Toolkit for Web Testing(SIT-WT)はWebアプリケーションの自動テストツールです。 テストスクリプトに従ってブラウザを自動操作し、エビデンスも自動で取得します。 テストスクリプトはExcelまたはCSVで作成可能で、プログラミングは不要です。 ブラウザの操作エンジンにはSelenium、Appiumを使用しているため、 EdgeやChrome、Firefox、iOS、Androidなど、Selenium、Appiumが対応している全てのブラウザをテストできます。 SIT-WTの詳細な情報はこちらを参照してください。

SIT-WTの詳細 クイックスタート

セットアップが簡単

SIT-WTはJavaがインストールされていればjarファイルをダウンロードしてダブルクリックするだけでテストを実行することができます。

動作可能なサンプルを提供

サンプルのテストスクリプトとWebページを提供しているため、操作方法や動作をすぐに試すことができます。

テストスクリプトの作成が簡単

テストスクリプトはExcelまたはCSVで作成します。作成にプログラミングスキルは不要で、データパターンの複製も簡単です。 Selenium IDEで記録したスクリプトから変換して作成することもできます。

可読性の高いエビデンスを自動取得

操作内容と操作項目に赤枠がついたスクリーンショットをエビデンスとして自動で取得するため、 手動テストで最も手間と時間のかかっていた作業を削減できます。

マルチブラウザ/マルチデバイス対応

ブラウザの操作にはSelenium、Appiumを使用しているため、 これらが対応しているブラウザ、デバイスでテストが実行できます。

並列実行に対応

コマンドにオプションを付けるだけで複数のテストを並列に実行することができます。 より多くのテストを短い時間で実行することができます。
