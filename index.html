SI-Toolkit for Application Development

SI-Toolkit for Application Development(SIT-AD)は継続的デリバリーを前提としたJava EE Webアプリケーション開発用のMavenアーキタイプです。 DBMSやAPサーバー、DBマイグレーションツール、自動テストツールが、インストール不要ですぐに使える状態でセットアップされています。 ツール群はMavenコマンド1つで実行できるため、JenkinsなどのCIツール上で動作させるのも簡単です。 SIT-ADの詳細な情報はこちらを参照してください。

SIT-ADの機能と特徴

SI-Toolkit for Web Testing

SI-Toolkit for Web Testing(SIT-WT)はWebアプリケーションの自動テストツールです。 テストスクリプトに従ってブラウザを自動操作し、エビデンスも自動で取得します。 テストスクリプトはExcelまたはCSVで作成可能で、プログラミングは不要です。 ブラウザの操作エンジンにはSelenium、Appiumを使用しているため、 EdgeやChrome、Firefox、iOS、Androidなど、Selenium、Appiumが対応している全てのブラウザをテストできます。 SIT-WTの詳細な情報はこちらを参照してください。
